# Advent of Code Solver

A simple Node.js application for solving Advent of Code problems.

## Overview
This is a straightforward Node.js app for solving Advent of Code puzzles. No frameworks, no dependencies - just vanilla JavaScript.

## Structure
- `solve.js` - Main solver file with part1() and part2() functions
- `input.txt` - Your puzzle input goes here
- `package.json` - Project configuration

## Usage
1. Paste your puzzle input into `input.txt`
2. Write your solution logic in the `part1()` and `part2()` functions in `solve.js`
3. Run with `npm start` or just `node solve.js`
